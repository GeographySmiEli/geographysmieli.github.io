NASA Meteorite Landings
  • Meteorite Landing Locations
  • Meteor, Meteoriod, or Meteorite?

For this project, I created an interactive map that visualizes historic meteorite remnants using Leaflet.js and jQuery. The map displays meteorite data, sourced from NASA's public API, with markers representing impact sites worldwide. Each marker's size is normalized to reflect the meteorite's mass. The meteorite data is fetched dynamically via an AJAX request, allowing the map to display up-to-date information without reloading the page.
